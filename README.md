# Installation et utilisation

## Installation
1. Clonez le dépôt :
   ```bash
   git clone https://github.com/DissidentAI/discord-server-configurator.git
   ```

2. Accédez au répertoire du projet :
   ```bash
   cd discord-server-configurator
   ```

3. Installez les dépendances nécessaires :
   ```bash
   npm install
   ```

## Utilisation
1. Démarrez l'application :
   ```bash
   npm start
   ```

2. Suivez les instructions à l'écran pour configurer votre serveur Discord.

## Aide
Pour toute aide supplémentaire, consultez la [documentation](https://github.com/DissidentAI/discord-server-configurator/wiki).
